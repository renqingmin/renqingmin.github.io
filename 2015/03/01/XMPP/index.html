XMPP环境配置

配置XMPP环境首先需要安装几个软件，（XAMPP，openfire 本地服务器，spark 模拟即时通讯软件）

一、安装XAMPP，一直next到底，最后安装完成关闭，出现如下界面

二、安装openfire服务器，一直继续到底，最后出现安装完成

三、在XAMPP中点击 Start All 启动所有服务，让所有服务全部 Running，如图

四、配置openfire服务器，在浏览器中输入 http://127.0.0.1:9090

（1）

（2）选择语言，根据自己擅长的语言选区，这里我们选择 中文（简体），然后点Continue

（3）服务器设置，设置域名，域名可以随便写，但不可以写中文和一些乱七八糟的符号。其余的默认不用修改，然后点击 继续

（4）数据库设置，两个数据库都可以用，建议用 标准数据库连接 ，然后点击继续

（5）数据库设置 －标准连接，数据库驱动选项 选择 MySQL，然后下面 JDBC 驱动程序类，数据库 URL 会默认自动填上，用户名 写 root，密码不写，然后暂停，当前页面不要关闭，再打开一个浏览器页面，见（6）

（6）配置XAMPP，在浏览器中输入 http://127.0.0.1

（7）然后选择，phpMyAdmin

（8）然后 New 创建一个数据库，输入一个数据库名字，然后点击创建，显示已创建数据库，创建成功

（9）选中刚创建数据库，会发现里面是空的，这时候我们需要向数据库中导入一个依赖库。首先我们要找到我们需要的依赖库，在跟目录下，选择 usr > local > openfire，这时候会发现openfire我们无权访问，右键openfire文件，点击 显示简介，然后将共享与权限全部改成 读与写，然后我们会看见openfire中的文件

（10）然后选择 openfire > resources > database > openfire_mysql.sql，然后将openfire_mysql.sql 拖到桌面

（11）回到配置XAMPP浏览器中，选中数据库，点击导入，选取文件，文件选择的是刚才我们拖到桌面上的 openfire_mysql.sql 文件，然后点击执行

（12） 回到配置 openfire服务器页面，在 数据库设置 - 标准连接 下对 数据库 URL 进行修改，将[host-name] 改成 127.0.0.1 ，[database-name] 改成刚才创建的数据库名字，然后点击继续

（13）特性设置，选择默认的 初始设置

（14）管理员帐户，直接跳过，可以不填，点击跳过。然后出现安装完成，最后点击登陆到管理控制台

（15）输入账号密码都是 admin

（16）登陆成功，会看见自己的服务器名称，此时XMPP环境已经配置完成
